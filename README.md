## 🇰🇷 한국어
본 저장소는 1,189장의 성경 전체를 **원어(히브리어/헬라어), 라틴어(불가타), 한국어(개역한글 현대화)**의 3단계 대조 구조로 재구성한 연구 자료실입니다.

### ✨ 주요 특징
*   **다국어 대조**: 개역한글의 깊은 맛을 살리면서 현대적 어휘로 정제.
*   **다원적 해석**: 가톨릭, 개신교, 동방정교회의 관점과 니체 등 철학적 통찰 대조.
*   **시각화**: Mermaid 다이어그램을 통한 성경 역사 도식화.

## 🇺🇸 English
A high-fidelity theological database reconstructing all 1,189 chapters into a **three-tier comparative structure: Original Languages, Latin (Vulgate), and Modernized Korean.**
